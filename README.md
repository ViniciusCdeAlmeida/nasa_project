
# APOD project

A simple Flutter Application that shows a list of Astronomical Pictures in chronological order fetched from NASA APOD API

## Rodando localmente

Clone the project

```bash
  git clone git@github.com:ViniciusCdeAlmeida/nasa_project.git
```

Run pub get

```bash
  flutter pub get
```

Run the project

```bash
  flutter run
```
